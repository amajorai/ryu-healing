# ryu-healing

Self-healing loop for Ryu — failed runs are diagnosed (Gateway side-model) and a fix is proposed to the approvals inbox.

> **Read-only mirror.** Developed in https://github.com/amajorai/ryu —
> please open issues and pull requests there, not on this repository.

## Install

- Binary: `ryu-healing` from the [Ryu releases](https://github.com/amajorai/ryu/releases).
- Crate: `cargo install ryu-healing`.
