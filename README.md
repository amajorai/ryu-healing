# ryu-healing

Self-healing loop for Ryu — failed runs are diagnosed (Gateway side-model) and a fix is proposed to the approvals inbox.

> **The public home of `ryu-healing`.** Source, builds, and releases live here —
> binaries for every platform are attached to each release.
>
> This tree is generated from the Ryu monorepo, so commits pushed here
> directly are replaced on the next sync. **Pull requests are welcome** —
> open them here and they are ported into the monorepo, then flow back out.
> Ryu as a whole: https://github.com/amajorai/ryu

## Install

- Binary: `ryu-healing` from the [Ryu releases](https://github.com/amajorai/ryu/releases).
- Crate: `cargo install ryu-healing`.
